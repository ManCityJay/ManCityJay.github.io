线性回归

  • 主要用途：预测连续的数值变量

  • 应用场景：房价预测、销售额预测、温度预测等任何输出是连续值的任务

  • 输出：可以是任何实数值（-∞到+∞）

逻辑回归

  • 主要用途：分类任务，特别是二分类问题

  • 应用场景：垃圾邮件检测、疾病诊断、客户购买预测、信用评分等

  • 输出：0到1之间的概率值，通常用阈值（如0.5）将其转换为类别标签

逻辑回归可以看作是在线性回归的基础上，添加了一个非线性转换（sigmoid函数）

随机梯度下降 minibatch 传统梯度下降的区别

  1. 随机梯度下降 (Stochastic Gradient Descent - SGD)：

    • 样本数量：每次参数更新只使用 一个 随机选择的训练样本来计算梯度。

    • 优点

      • 更新速度快，因为每次迭代的计算量非常小。

      • 由于梯度的随机性，它有时能够跳出局部最优解，从而可能找到更好的全局最优解。

    • 缺点

      • 梯度估计的方差很大，导致损失函数的下降过程非常嘈杂，收敛路径曲折。

      • 无法充分利用现代计算硬件（如 GPU）的并行计算能力。

  2. 小批量梯度下降 (Mini-batch Gradient Descent)：

    • 样本数量：每次参数更新使用一 小批 (mini-batch) 随机选择的训练样本（例如 32, 64, 128 个样本）来计算梯度。梯度是这一小批样本梯度的平均值。

    • 优点

      • 与 SGD 相比，梯度估计的方差更小，损失函数的下降过程更平稳，收敛更稳定。

      • 可以利用矩阵运算的优势，从而更有效地利用硬件的并行计算能力，训练速度通常比 SGD 更快。

      • 是 SGD 和批量梯度下降 (Batch Gradient Descent) 之间的一个折中方案。

    • 缺点

      • 需要额外调整一个超参数：批量大小 (batch size)。

      • 虽然比 SGD 稳定，但梯度仍然存在一定的噪声。

卷积神经网络 相较于 全连接前馈神经网络的优势是什么？

填充的意义：

  1. 保持特征图的空间维度

  2. 更好地利用图像边界信息：图像边缘的像素被卷积核扫过的次数远少于图像中心的像素。这意味着网络可能无法充分学习到图像边缘的特征。

3个输入通道，1个卷积核就变成a×b×3，一个卷积核必须能处理所有输入通道（3维卷积核）

卷积核的个数 = 输出层通道数（一个通道即一个特征图，输入可能是上一层卷积网络，特征图/通道数完全可以大于3）

多个卷积核提取多种特征，增强卷积层能力

对于每一个卷积核，在计算完它与输入数据的加权和之后，通常还会加上一个偏置项 b（计算参数量注意，但PPT中又没有明说）

pooling

  1. 在连续的卷积层之间周期性地插入池化层是一种常见的做法。

  2. 主要功能

    • 逐步减小表示的空间尺寸：池化层会缩小特征图的高度和宽度。

    • 减少参数量和计算量：通过减小特征图尺寸，后续层的参数和计算需求会降低。

    • 控制过拟合：减少参数和计算量也有助于防止模型在训练数据上过度拟合。

    • 但同时保留了最重要的特征信息。

  3. 工作方式

    • 池化层对输入特征图的每一个通道独立进行操作。

    • 它只改变特征图的宽度和高度，不改变深度（通道数）。

  4. 最常见的形式举例

    • 2x2大小的滤波器，步长为2（垂直方向也是移动2像素）：这是最常见的池化配置。

    • 它会将输入的每个深度切片（通道）的宽度和高度都缩小一半（下采样因子为2）。

    • 丢弃75%的激活值：因为在一个2x2的区域（4个值）中，只取一个最大值/平均值，所以75%的激活值被丢弃了。

    MAX操作的具体行为：在这种情况下，每个MAX操作都是从4个数字（某个深度切片上的一个2x2小区域）中取最大值。

    最大池化：从每个区域中选取最大的激活值。它倾向于保留最显著的特征。

    平均池化：计算每个区域内激活值的平均值。它倾向于保留区域内特征的整体平滑信息。

降采样(DownSample)
(1)减少计算量,防止过拟合
(2)增大感受野,使得后面的卷积核能够学到更加全局的信息。
例如，输出特征图上的一个像素点是由输入特征图上一个2x2的区域计算得来的（取最大值或平均值）。这意味着，降采样后的特征图上的一个像素点，已经“代表”或“概括”了它在降采样之前对应的那片更大区域的信息。

增大感受野
增加卷积核的大小 增加层数来实现 在卷积之前进行池化操作
